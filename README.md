# Cultivation World / 修仙游戏

当前版本：洞府点击升级原型。

## 已完成

- 单页游戏入口：`index.html`
- 高级洞府成品图作为主视觉
- 主殿、灵田、丹炉、灵池、玉桌、灵灯、仙鹤雕像透明点击热区
- 建筑升级弹窗
- 灵气、灵石、境界显示
- 修炼一次、收取收益、重置存档
- `localStorage` 本地保存：`cultivation-world-save`
- 中文 / English 切换
- 热区显示/隐藏，便于调点击范围

## 设计原则

当前阶段不做自由拖拽建造。先使用「成品府邸图 + 透明点击热区 + 升级系统」稳定落地。后续流量起来后，再把洞府升级为自由建造系统。

## 运行方式

直接打开 `index.html` 即可。

后续可接入 GitHub Pages / Cloudflare Pages，变成可手机直接访问的网页游戏。
